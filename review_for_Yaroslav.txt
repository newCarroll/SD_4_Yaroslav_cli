Недочеты
1) Хотелось бы какого-то приветствия при начале работы,
   а то непонятно запустилась программа или еще загружается
2) При введении ерунды в качестве команды выдется сообщение:
   Error: Cannot run program "fgrehjg": error=2, No such file or directory
   Мне кажется достаточно оставить только Error: Cannot run program "fgrehjg", 
   либо изменить пояснения, например, на "нет такой команды"
3) какие-то проблемы с shell, а именно при вводе команды, которая есть в shell,
   но нет в интерпретаторе не выводится ошибки (если произошла ошибка), а просто
   интепретатор дальше работает, да и при введение неошибочной команды не выдает результат

Плюсы
1) Проект хорошо структирирован, все названия понятны, не было проблем с поиском того или иного компонента
2) В целом есть комментарии, при этом они понятны, их немного - ничего не загромаждают и по делу
3) Функциональность в целом выполнена, единстевенный недочет - это команды для shell
4) понравился класс Lexer для анализа и деления на токены (а не как отедльная функция как у меня), но что-то страшное творится с if-else при подставлении значений переменных
